# Activity Planner

A modern, lightweight PWA application for planning activities with integrated todo lists.

## Features

- **Monthly Timeline View**
- **Integrated Todo Lists**
- **Two-Click Delete Protection**
- **Offline Support (PWA)**
- **Local Storage** - Also support images storage
- **Responsive Design**

## Live Demo

[**https://rduvql.github.io**](https://rduvql.github.io)

## Built With

- Vue 3
- TypeScript
- Vite
- Tailwind CSS
- Vite PWA Plugin

## Installation
```bash
# Install dependencies
pnpm install

# Run development server
pnpm run dev

# Build for production
pnpm run build
```

## Data Storage

All data is stored locally in your browser's localStorage. No server required, all data stay on device
